About This Project

// PROJECT OVERVIEW

Terminal OS is a retro-inspired, terminal/hacker aesthetic website built with modern web standards. It combines nostalgic computing elements with accessibility compliance and interactive features.

// FEATURES

  • BIOS-style boot sequence with system checks
  • Typing animation with blinking cursor
  • ASCII art headers and decorations
  • CRT scanline overlay (toggleable)
  • Matrix rain background effect (toggleable)
  • Dual theme support (Dark Neon / Accessible Dark)
  • Command palette-style search (Ctrl/Cmd + K)
  • Full CLI terminal with 10+ commands
  • Classic Snake game with high score tracking

// TECHNICAL SPECS

  • Pure HTML, CSS, and vanilla JavaScript
  • No external libraries or frameworks
  • WCAG 2.1 AA accessibility compliant
  • Respects prefers-reduced-motion
  • Mobile responsive design
  • Keyboard navigable

// TYPOGRAPHY

Primary: JetBrains Mono (Apache 2.0)

Secondary: Fira Code (SIL OFL)

// COLOR THEMES

Dark Neon: #0a0a0a bg, #00FF00 text, #FF00FF accent

Accessible Dark: #121212 bg, #E0E0E0 text, #76FF03 accent
